### System Topics
- **Kernel Subsystems**: Managing processes, memory, and devices.
- **User Space vs Kernel Space**: The isolation between user applications and the core OS.
- **Boot Process**: BIOS/UEFI → Bootloader (GRUB) → Kernel → Init/systemd.
- **Filesystem Structure**: The hierarchical layout (FHS) from root (`/`).
- **systemd**: The standard system and service manager for modern Linux distributions.
