# Red Team Tools

## Summary
A compilation of essential tools used in red team engagements for reconnaissance, scanning, and exploitation.

## Details
Key tools categorized by their primary use case:
- **Reconnaissance**: subfinder, amass, Google dorking, waybackurls.
- **Scanning & Fuzzing**: nmap, nikto, nuclei, ffuf, gobuster.
- **Exploitation**: Burp Suite, sqlmap, jwt_tool.
